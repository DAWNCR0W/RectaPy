<h1 align="center">RectaPy</h1>

> Python Implementation of Recta Language

## 📖 Prerequisites

* python >= 3.7

## 🚀 Usage

### Run from a file

> python -m rectapy [filename]

### Run a prompt

> python -m rectapy
